# 📊 Amazon Data Analysis Dashboard (Power BI)

---

## 📌 Project Overview
This project presents an *interactive Power BI dashboard* analyzing Amazon sales data.  
The dashboard provides insights into *orders, revenue, quantity sold, customer ratings, product categories, payment methods, discounts, and regional distribution*.

It helps stakeholders understand:
- Sales performance
- Customer purchasing behavior
- Product category trends
- Payment method contribution
- Discount impact on sales
- Regional distribution of customers
